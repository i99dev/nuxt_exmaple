# Nuxt3 Practice
## 1. Nuxt3 install

```bash
$ npx nuxi init <project-name>
```

## 2. Nuxt3 run

```bash
$ npm run dev
```

## 3. Nuxt3 build

```bash
$ npm run build
```

## 4. Nuxt3 start

```bash
$ npm run start
```

## 5. Nuxt3 generate

```bash
$ npm run generate
```

Nuxt3 generate static files in `dist` folder. and you can deploy it to any static hosting service.
